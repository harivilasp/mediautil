# Changelog

All notable changes to `mediautil` will be documented here.

## 0.1.1 - Unreleased

### Changed

- Improve public package descriptions and installation instructions across README, crates.io, npm, PyPI, and Homebrew metadata.
- Document live validation against real OCR and PDF tools.

## 0.1.0 - 2026-07-17

### Added

- Rust CLI with image, PDF, OCR, QR, data URI, and base64 command groups.
- Native image resize, crop, convert, icon, data URI, QR, and base64 workflows.
- External adapters for Tesseract, Poppler `pdftotext`, `qpdf`, `pdfcrop`, MuPDF `mutool`, and ImageMagick.
- Functional CLI test suite with temporary files and fake external binaries.
- GitHub Actions CI, release workflow, Dependabot, package scaffolding, and contributor/security docs.
